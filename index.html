Vantage

CSE 30246

Application Domain

Vantage is an application that provides a comprehensive overview of the remote machines available to students within the Notre Dame network, namely the CSE student machines and the CRC machines. Using Vantage, users will be able to get historical, detailed analysis of the use of computing resources, as well summary reports via email and/or slack. Advanced search functionality will allow administrators to view statistics based on individual users, given dates, etc.

Usefulness

Currently, there are no web applications that provide real-time and historical data on the usage of remote machines on Notre Dame's network. This information would be helpful for administrators, professors, and even students, allowing them to understand who is using the resources and for what purpose. Furthermore, CSE professors have shown an interest in having this type of application available to them.

Realness

We plan on writing several scripts that gather important data on the student machines and then write it to our database. The scripts will run periodically using cron jobs (say every 5 minutes). If these scripts begin running by early October, we will have two months worth of historical data to use in our application.

Planned Functionality

Basic Functions: Vantage will support the ability to insert, delete, and update records in the database. Most of this functionality will be utilized by our scripts that gather the data from the machines.

Advanced Functions: First, Vantage will have an algorithm that uses the historical data to predict when resources are under the greatest strain, allowing users to have the insights needed to log in to different machines or do work at different times. Second, Vantage will generate a weekly report that summarizes remote machine usage. This feature will give the people in charge of monitoring the computing resources the information they need in a convenient way.
